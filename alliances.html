Alliances/Technology Partners

Synergy Software is an IBM BusinessPartner in several areas including AS400, SanFrancisco Frameworks , WebSphere and RS6000 technologies. Synergy has developed award winning software for IBM platforms.

BEA SYSTEMS
Synergy Software is a technology partner with the BEA Systems utilizing BEA's WebLogic family of application servers

ORACLE
Synergy Software is an oracle alliance partner developing applications utilizing all Oracle Tools. Synergy's Freight Management System is fully compatible with Oracle Applications .

PROGRESS
Synergy Software is a licensed Progress Alliance Partner. Synergy's Freight Forwarder System is developed on Progress platform

Synergy Software is a licensed Microsoft applications developer. Synergy's Microsoft certified engineers and programmers have developed mission critical applications using whole array of Microsoft OS, database, and language suites.
