Synergy helps local church with the web presence

.   St Peters and St pauls church was looking for a solution to handle two different tasks 1) To reach out to the computer savvy group of youngsters and provide opportunities for them to take part in discussions, contests, and other activities 2)to allow the members to make pledges and manage their contributions

Charity organizations have unique requirements which are not met by the standard systems. Synergy's charity software has two specific areas where typical systems do not offer any support at all

1. Provide a robust web frontend with full featured text management features allowing the organization to publish material for its members and the general public. Synergy's Sitemaker provides the framework which can be used to create several different types of sites.

These informational sites come with search features as well as several newsgroup like functions with areas for community, faith, organization related matters and so on.

2. Prvide a robust backend system with
membership managemnt
event calendar
pledge accounting
material/gift accounting
auction/asset sale support
individualized statements
organization budget, and tax forms support
online analysis features

St Peters and St Pauls Church, Sparkill has already started using the features in a test mode. Individual signon for all parish members are provided which allow them to pledge for specific causes and get their tax-recipts and statements.

Visit St Peters Church site at www.stpetersandstpaulschurch.com.
