Synergy to use LANSA for the AS400

.   Synergy Software signed an agreement with LANSA to use LANSA's suite of web development products for the AS400. The AS/400 is the most popular commercial transaction system and Windows-based PC's are the most popular on the desktop. As user departments share information, business solutions must provide Windows user interface consistency and an AS/400 standard of commercial stability across a diverse range of platforms, including UNIX and Windows NT. Your LANSA applications smoothly port across platforms including browsers and Java. With LANSA, you can transform an AS/400 RPG package into a portable and highly graphical client/server application. Your LANSA applications can run on other platforms including Windows 95/98, Window NT/2000, UNIX and the Web. You can rapidly develop an e-business Internet, Extranet, or Internet extension for your existing application using LANSA for the Web. LANSA lets you create instant client/server demonstrations with our query and reporting tool. Only with LANSA can you add client/server capability with our choice of Windows tool including PowerBuilder, Visual Basic or LANSA. Our fast scalable middleware ensures rapid response and ease-of-use. Of course you can redevelop your application entirely in LANSA for optimal portability. Maintain one set of source code and let LANSA take care of multiple platforms, databases and network complexities. LANSA's Openness means you can always mix the high productivity and maintainability of portable LANSA applications with complementary technologies such as the Microsoft Desktop, Java, Lotus Notes, or Mobile computing. A single version of a LANSA program can support multiple languages including DBCS (for the North Asian market) and RLTB ( for the Middle East).

See more information on LANSA at www.lansa.com.
